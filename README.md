## 소개
신발 쇼핑몰 웹 서비스의 백엔드 입니다.  
Spring Boot 기반 REST API 서버입니다.

## 기술 스택
- Java
- Spring Boot
- JPA
- Postgres DB (SupaBase)

## 주요 기능
- 회원가입 / 로그인
- 상품 목록 조회
- 상품 상세 페이지
- 장바구니
- 주문 기능
- 관리자 상품 관리
